🎯 导航居中测试

这个页面用于测试导航菜单是否正确居中显示。

✅ 如果导航菜单（首页、AI工具、AI模型、AI代理）显示在页面中央，则测试成功！
📍 请检查上方导航栏中的菜单项是否居中对齐

📋 测试检查清单

  • ✅ Logo "MyAI" 应该在左侧
  • ✅ 导航菜单（首页、AI工具、AI模型、AI代理）应该在中央
  • ✅ 移动端菜单按钮应该在右侧（小屏幕时）
  • ✅ 导航菜单项之间有适当的间距
  • ✅ 当前页面的菜单项应该有高亮显示

🔧 CSS 修改说明

为了实现导航居中，我们修改了以下CSS：

.main-header .container {
    max-width: 1200px;
    margin: 0 auto;
    padding: 0 1rem;
    display: flex;
    justify-content: center;  /* 改为居中 */
    align-items: center;
    position: relative;       /* 添加相对定位 */
}

.main-header .logo {
    position: absolute;       /* 绝对定位logo到左侧 */
    left: 1rem;
}
